# Little Lemon - Course 7
Welcome to the **Little Lemon** project, developed as part of course 7. 
This web application represents a modern Mediterranean restaurant, focused on providing fresh, authentic, and elegant culinary experiences.
## Features
- Presentation page with the restaurant's values: Freshness, Passion, and Community
- Contact section with functional form- Responsive and elegant design
- Optimized and aligned images
- Meta tags for SEO and social media (Open Graph)
## Technologies used
- React.js
- HTML5 + CSS3
- Git + GitHub
- Visual Studio Code
